# スーパーマリオ風ブラウザゲーム

このプロジェクトは、HTML5 と Phaser フレームワークを使用して作られたスーパーマリオ風の 2D プラットフォームゲームです。
リンクはこちらから　https://helpmark-pink.github.io/mario-pakuri/

## 特徴

- 5 つのユニークなステージ

  - ステージ 1: 基本的なプラットフォームステージ
  - ステージ 2: 夜のステージ（動く足場）
  - ステージ 3: ピンク空の空中戦ステージ
  - ステージ 4: 地下ステージ
  - ステージ 5: 最終城ステージ

- 多彩なゲームシステム

  - コイン収集システム
  - スコアシステム
  - ライフシステム
  - パワーアップアイテム（キノコ、ファイアフラワー、スター）

- 多様な敵キャラクター

  - クリボー
  - ノコノコ
  - ハンマーブロス
  - スピニー
  - ジュゲム
  - パックンフラワー

- レスポンシブデザイン
  - PC とモバイル両方に対応
  - タッチスクリーンコントロール対応

## 必要条件

- モダンな Web ブラウザ（Chrome, Firefox, Safari, Edge 等）
- インターネット接続（Phaser ライブラリの読み込みに必要）

## 遊び方

1. `index.html`をブラウザで開く
2. スペースキーまたは画面タップでゲーム開始
3. 操作方法:
   - PC: 矢印キーで移動、スペースキーでジャンプ
   - モバイル: 画面上の矢印ボタンで移動とジャンプ

## 開発環境のセットアップ

1. リポジトリをクローン:

```bash
git clone [リポジトリURL]
```

2. プロジェクトディレクトリに移動:

```bash
cd マリオ
```

3. ローカルサーバーを起動（例：Python3 の場合）:

```bash
python -m http.server
```

4. ブラウザで `http://localhost:8000` にアクセス

## クレジット

- フレームワーク: [Phaser 3](https://phaser.io/)
- アセット: すべてのゲーム内グラフィックスはプログラムで生成
